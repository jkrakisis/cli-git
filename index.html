파일생성

git command list

git init
git 저장소 초기화
git status
git 상태
git add[file, *]
버전관리 스냅샷 생성
git commit -m "메모내용"
영구적으로 버전관리 스냅샷 파일기록
git diff
스테이지 되지 않은 변경사항을 화면에 출력
git diff --staged
스테이지된 이후, 변경사항을 화면에 출력
git log
커밋된 결과물으 화면에 출력
git reset HEAD {file}
파일 언스테이지드(Unstaged)
git reseet Head^
최근 커밋 취소
git checkout -- {file}
파일이전 커밋 스냅샷 기록으로 복귀
git remote add {저장소 이름 } "원격저장소 경로"
원격저장소 등록
git remote
원격저장소 정보 출력
git remote -v
원격저장소 정보 출력
git push {원격 저장소 이름} {브랜드 이름}
로컬 git 저장소 데이터 원격 저장소로 업로드
